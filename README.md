## 🚀 Quick Start Guide

### Prerequisites
```bash
Node.js >= 14.17.0
MongoDB >= 4.4
npm >= 6.14.0
```

### One-Command Setup
```bash
# Clone and install
git clone https://github.com/Alfayads/Yadro.git && cd Yadro && npm install

# Configure environment
cp .env.example .env

# Launch development server
npm run dev
```
